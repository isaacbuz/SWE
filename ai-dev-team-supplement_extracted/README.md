# AI Dev Team — Supplemental Docs

**Date:** 2025-11-08

This package consolidates the design findings, diagrams, and prompts for the **AI Dev Team** platform-leveraging architecture (GitHub-first, MCP-integrated, Mixture-of-Experts).

**Folders**

- `/docs` — architecture, agents, MoE routing, MCP tools, GitHub integration, quality gates, observability, costs.
- `/diagrams` — Mermaid + PlantUML text you can paste into docs; ASCII sketches.
- `/prompts` — production-ready prompt for Claude (and others) to implement in your IDE.
- `/samples` — tiny reference snippets: config, Streamlit stub, MCP manifest examples.
- `/sequences` — step-by-step sequence narratives for key flows.
