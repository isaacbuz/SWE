## Quick Start for Contributors

1. **Pick an issue**: Browse open issues, filter by label
2. **Assign yourself**: Comment to claim the issue
3. **Create branch**: `git checkout -b feature/issue-XX-description`
4. **Implement**: Follow acceptance criteria
5. **Write tests**: Maintain 80%+ coverage
6. **Create PR**: Reference issue number
7. **Code review**: Address feedback
8. **Merge**: After approval + green CI
